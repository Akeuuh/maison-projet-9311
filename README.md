## How to Rebuild

```bash
cd site/
node build.js
```

This regenerates the `dist/` folder from source files.

## How to Deploy

```bash
git add dist/
git commit -m "update"
git push origin main
```

Changes will be live at the GitHub Pages URL within a few minutes.

## Site Contents

- **index.html** - Dashboard overview
- **annonces/** - Real estate listings analysis
- **projet.html** - Project summary
- **financement.html** - Financing details
- **maison-chilly.html** - Chilly property details
- **appart-clichy.html** - Clichy apartment details

## Configuration

Hosted on GitHub Pages, served from the `dist/` folder on `main` branch.
